# Vktun.IoT.Connector.Protocol

工业协议解析器，支持 Modbus RTU/TCP、IEC104、S7、OPC UA、BACnet、CANopen 等多种工业协议。

## 安装

```bash
dotnet add package Vktun.IoT.Connector.Protocol
```

## 功能

- Modbus RTU/TCP 协议解析
- IEC104 协议解析
- S7 协议解析
- 自定义协议支持
